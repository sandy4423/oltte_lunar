- **2.5(목) 22:00**: 환영 팝업 표시 → 정상 주문 가능 → PAID
- **2.5(목) 23:30**: 추가 주문 팝업 표시 → 추가 주문 가능 → LATE_DEPOSIT (정상 배송)
- **2.6(금) 05:00**: 추가 주문 팝업 표시 → 1시간 남음 → LATE_DEPOSIT (정상 배송)
- **2.6(금) 07:00**: 픽업 전용 팝업 표시 → 온라인 주문 불가
- **2.11(수)까지**: 기존 발급된 가상계좌로 입금 가능

### 중요 안내

- **LATE_DEPOSIT(마감 후 입금)**: "늦은 주문"이 아니라 공식 마감(D-1 23:00) 이후 입금을 의미합니다
- **추가 주문 기간**: 마감 후에도 배송일 06:00까지는 정상적으로 주문하고 배송받을 수 있습니다
- **가상계좌 유효기간**: 배송일 +5일까지 유효하므로 여유롭게 입금하실 수 있습니다
